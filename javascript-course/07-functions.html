What is function?

  • A function lets is reuse code
  • Function creates scope
  • Function makes it easier to read

Rules for function names:

  • Can't use special words
    Example: function
  • Can't start with a number
  • Can't use special characters Except: $ and _
  • Best practice is to use camelCase for function names

Best Practice of creating function

  • Use camelCase for function names
  • Use a verb/action at the beginning of function name
  • Keep variable inside a scope and use return value
