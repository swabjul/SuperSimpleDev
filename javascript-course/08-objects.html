Rules for function names:

  • Can't use special words
    Example: function
  • Can't start with a number
  • Can't use special characters Except: $ and _
  • Best practice is to use camelCase for function names

Best Practice of creating function

  • Use camelCase for function names
  • Use a verb/action at the beginning of function name
  • Keep variable inside a scope and use return value

7d. Create a function 'convertToFahrenheit(celsius)' that takes a number in degrees Celsius and returns a number in degrees Fahrenheit.

  • Formula: Fahrenheit = (Celsius * 9/5) + 32
  • convertToFahrenheit(25) => 77
          function covertToFahrenheit(celsius) {
            return (celsius * 9/5) + 32;
          }
          
          console.log(covertToFahrenheit(25));

          // 77

Create a function 'convertToCelsius(fahrenheit)' that takes a number in degrees Fahrenheit and returns a number in degrees Celsius

  • Formula: Celsius = (Fahrenheit - 32) * 5/9
  • convertToCelsius(86) => 30
          function convertToCelsius(fahrenheit) {
            return (fahrenheit - 32) * 5/9
          }
          
          console.log(convertToCelsius(86));

          // 30

Create a function 'convertTemperature(degrees, unit)' that takes a number and a unit ('C' or 'F'), and converts it into the other unit.

  • convertTemperature(25, 'C') => '77F'
  • convertTemperature(86, 'F') => '30C'

Note: return a string, and try to reuse the function from 7d and 7e.

          function covertToFahrenheit(celsius) {
            return ((celsius * 9/5) + 32);
          }
          
          function convertToCelsius(fahrenheit) {
            return ((fahrenheit - 32) * 5/9);
          }
          
          function convertTemperature(degrees, unit) {
          
            if (unit === "C") {
              const fahrenheit = covertToFahrenheit(degrees);
              return fahrenheit + "F";
          
            } else if (unit === "F") {
              const celsius = convertToCelsius(degrees);
              return celsius + "C";
            }
          }
          
          console.log(convertTemperature(25, "C"));
          console.log(convertTemperature(86, "F"));

          // 77F
          // 30C
