SuperSimpleDev- Objects -

What is an object

  • An object groups multiple values together

Dot Notation

  • Object.property

Why we use objects

  • Makes our code more organized

Ways of calling value of object

  • Dot Notation
  • Bracket Notation - lets us use properties that don't work with dot notation

Function inside object

  • This is called method

JSON Built-in Object

  • Help us work with JSON
  • Javascript Object Notation
  • similar to JS Object but with less features

We use JSON

  • When we send data between computers
  • when we store data

Built-in JSON Object

  • Javascript Object => JSON (JSON.stringify();)
  • JSON => Javascript Object (JSON.parse();)

localStorage

  • Save values permanently
  • localStorage.setItem();
  • localStorage.getItem();
  • localStorage.removeItem();

Auto-boxing

  • Javascript automatically wrap ang string,numbers and booleans into an object like a box
  • Auto-boxing doesn't work with null and undefined result in error

          const product1 = {
            name: "basketball",
            price: 250,
          }
          
          const product2 = {
            name: "football",
            price: 250,
          }
          
          function comparePrice(product1, product2) {
          
            if (product1.price < product2.price) {
              return product1;

              return product2;
