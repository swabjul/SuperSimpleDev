Asynchronous Code

Synchronous Code

.forEach()

  • This allows to loop through an array

.addEventListener()

  • Lets us run some code when we interact with the element
  • Can add multiple listeners in one event
  • We can remove event listeners

.filter()

  • Creates a new array []
  • return true, => put the value in array
  • return false, => not put the value in array

.map()

  • Create a new array []
  • Whatever we return => add to new array
  • Transform an array into another array

Closure

  • If a function has access to a value
  • It will always have access to that value
  • Value gets packaged together (enclosed) with the function

12l. Create a function countPositive(nums) that takes an array of numbers and returns how many numbers in the array are greater than 0. Create this function using .forEach() instead of a loop

  • countPositive([1, -3, 5]) => 2
  • countPositive([-2, 3, -5, 7, 10]) => 3
          function countPositive(nums) {
            let count = 0;
          
            nums.forEach((value) => {
              value >= 0 ? count++ : count;
            });
            
            return count;
          }
          
          console.log(countPositive([1, -3, 5]));
          console.log(countPositive([-2, 3, -5, 7, 10]));

12n. Create a function removeEgg(foods) that takes an array of strings and returns an array where the string 'egg' is removed. Create this function using .filter() instead of a loop

  • removeEgg(['egg', 'apple', 'egg', 'egg', 'ham']) => ['apple', 'ham']
          function removeEggs(foods) {
            return foods.filter(value => value !== "egg");
          };
