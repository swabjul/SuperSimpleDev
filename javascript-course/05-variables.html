SuperSimpleDev- Variables -

What is a variable?

  • Variable is a container that we can save a value and then use it later

Variable name restrictions

  • Can't use special words. Example: let
  • Can't start with a number
  • Can't use special character. Except $ and _

Best practices for naming variable

  • camelCase
  • Use const by default (Only use let when we need to change the variable)

Naming Convention

  • camelCase - is the stand for JavaScript
  • PascalCase - use in 1 feature in JavaScript
  • kebab-case - doesn't work in Javascript but we can use it in HTML and CSS
  • snake_case - used in other language

3 way creating a variable

  • let = if we have to change value
  • const = can't change its value
  • var = we don't use this in new JavaScript
