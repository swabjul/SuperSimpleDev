What are Booleans?

  • Booleans are another type of value
  • There are only 2 boolean values: true and false

What are the purpose of booleans values

  • A boolean value represents whether something is true or false

Comparison Operators

  • > greater than
  • < less than
  • >= greater than or equal to
  • <= less than or equal to
  • === equal to
  • !== not equal to

Order of Operation (Update)

  • (...)
  • * /
  • + -
  • Comparison Operators
  • Logical Operetaors

If=Statements

  • Lets us write multiple groups of code
  • And then decide which code to run

Logical Operators

  • && AND Operator
  • || OR Operator
  • ! NOT Operator

Falsy Values

  • false
  • 0
  • ''
  • ""
  • NaN
  • undefined
  • null

Shortcuts for If-Statements

  • Ternary Operator ?:
  • Guard Operator &&
  • Default Operator ||

Ternary Operator

  • condition ? exprIfTrue : exprIfFalse
  • We can the ternary operator in a variable

Guard operator

  • This is the same as AND operator but can be assigned in a variable
  • const var = true && statement;

Default Operator

  • This is the same as OR operator but can be assigned in a variable
  • const currency = undefined || 'USD'
  • We can use or operator for default value
